Lien du site web AniShop: https://jz1025.github.io/Site-Narratif/ 

Ce projet nous a inspiré a faire un site web pour magasiner des articles de presques tous les animes japonais qui existent. Nos animes préférées sont Chainsaw Man, Zom 100, Dr. Stone, Demon Slayer, RE:0 et Akira. Ce site a cinq pages. Une page qui est pour les cartes de collections, une page qui est dédié pour les figurines, une page d'acceuil, une page pour électronique et gaming et une page pour les peluches. Dans ce site web, il y a quatre catégories d'articles qui permet d'avoir une varitété de choix pour notre magasin web AniShop.

Moi et Jingze ont ait honnête avec vous, nous avons utilisé l'IA pour corrigez nos codes, puisqu'il y a avait des templates trop comnpliquer à faire pour notre TP. Cependant, moi et Jingze ont comprend que cet outil est seulement pour aider à notre apprentissage et par la suite pour que nous devenions meilleur. On a tout simplement posé des questions, comme "Pourquoi mon css ne marche pas ? et "Pourquoi il y a des images plus grosses que les autres ?"
